Este é um projeto pessoal/profissional, meu primeiro projeto real.

O objetivo deste projeto é criar um site para uma decoradora. Nele, o cliente pode alugar decorações, fazer pedidos personalizados, como Bolos Fake e Decorações Completas e comprar itens para suas festas, como pratos, talheres e copos descartáveis.

O projeto é feito com HTML, CSS, JavaScript, Bootstrap e FontAwesome.

O site possui um carrinho de compras. Para finalizar o pedido, o site redireciona para o WhatsApp, junto à um texto personalizado com os produtos do carrinho.

O projeto não utilizado banco de dados, as informações necessárias são armazenadas no localStorage.
